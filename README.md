# Test Project with NestJS and Prisma

This is a test project built with NestJS and Prisma, demonstrating a simple user and product management system with authentication using JWT. The project includes basic CRUD operations products, as well as user authentication and registration.

## Features

- User registration and authentication
- JWT-based authentication
- CRUD operations for products
- Pagination for product listings
- Prisma ORM for database interactions
- Integration with Swagger for API documentation

## Getting Started

### Prerequisites

- Node.js (>= 14.x)
- npm (>= 6.x) or yarn (>= 1.x)
- PostgreSQL (or any other database supported by Prisma)

### Installation

1. Clone the repository:

```sh
git clone https://github.com/EdvardSolomon/Products-api
cd products-api
```

2. Install dependencies:

```sh
npm install
# or
yarn install
```

3. Set up the environment variables:
   Create a copy of the .env.example file and rename it to .env. Update the file with your own values:

```sh
cp .env.example .env
```

Edit the .env file with your database credentials and other configuration values.

4. Run Prisma migrations:

```sh
npx prisma migrate dev
```

Running the Application
Start the development server:

```sh
npm run start:dev
# or
yarn start:dev
```

The server will start on http://localhost:3000.

API Documentation
You can access the API documentation generated by Swagger at http://localhost:3000/api.

Project Structure
src: Contains the source code of the application
auth: Authentication module (DTOs, guards, services, controllers)
common: Common utilities and decorators
prisma: Prisma service for database interactions
products: Product module (DTOs, services, controllers)
users: User module (DTOs, services, controllers)
test: Contains test files for the application

Running Tests
To run the tests, use the following command:

```sh
npm run test
# or
yarn test
```
